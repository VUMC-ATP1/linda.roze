## Home Work 1
### Par mani
Mani sauc Linda. Pašlaik apgūstu automatizēto testēšanu kursā ar nosaukumu **Automatizēta programmatūras testēšana**
Strādāju IT jomā kā testētājs. Mana līdzšinējā praktiskā pieredze šajā jomā ir tikai manuālo testu veikšana. Esmu piedalījusies un organizējusi lietotāju apmācības. Ir pieredze arī prasību analīzes jomā.
